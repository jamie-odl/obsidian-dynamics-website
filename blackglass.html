FLEET CONSOLE

Blackglass

A Next.js 16 multi-tenant fleet console for baselines, drift, evidence exports, and Stripe billing hooks — with Clerk auth, Drizzle ORM and PostgreSQL, and DigitalOcean-ready deployment. The canonical product repository is public on GitHub.

REPOSITORY

Align your deployment with upstream

Use the same source tree for local development, CI (verify:stage0), and production builds as described in the project README.

Clone

git clone https://github.com/thevenomv/blackglass-console.git

Requirements

Node.js 22.x and npm 10+. Copy .env.example to .env.local and follow operator docs in the repo for Clerk, database, and Stripe.

Baselines and drift

Govern baseline posture and drift visibility across fleet inventory from the authenticated dashboard.

Evidence exports

Export-grade artefacts and audit-friendly flows aligned with tenant isolation (RLS) in the application database.

Billing

Stripe checkout, webhooks, and plan limits (plans.ts) stay in sync with subscription state idempotently.
