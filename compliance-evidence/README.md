# Compliance Evidence Pack

This folder is the centralized archive location for SOC 2 Type II and GDPR evidence artifacts.

## Included Records

- `QUARTERLY_ACCESS_REVIEW_2026Q2.md`
- `RESTORE_DRILL_LOG_2026Q2.md`
- `RETENTION_DELETION_AND_DSAR_EVIDENCE.md`
- `DPA_SCC_COVERAGE_TRACKER.md`
- `RENDER_DECOMMISSION_EVIDENCE.md`

## Usage

1. Complete each record with reviewer names, dates, and evidence links.
2. Attach exported system reports/screenshots into your controlled evidence store.
3. Reference those immutable artifacts in the "Evidence reference" columns.
4. Obtain final sign-off from security/compliance owners per record.
