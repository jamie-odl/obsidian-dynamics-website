UNIFIED API

One API. Clear surfaces.

SkyGrid and the Blackglass console follow consistent operational patterns: shared conventions where the parent platform exposes air risk events, and a separate in-repo OpenAPI for the fleet SaaS.

Endpoints

GET /v1/air/events
GET /v1/maritime/events
GET /v1/nodes/{id}/risk
GET /v1/history/replay

Blackglass ships its own OpenAPI surface in the blackglass-console repository (check:openapi in CI).

Common Conventions

Same auth, pagination, timestamp format, severity and confidence vocabulary, and billing model across all modules.
