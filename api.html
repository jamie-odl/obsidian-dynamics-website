API & INTEGRATIONS

One vocabulary. Scoped access.

Nodes talk to the platform over a small, stable HTTP surface. Operators get a keyed API and metrics; auditors get exports. Full reference and keys ship with onboarding.

Representative routes: POST /api/ingest (batched readings, returns 202), POST /api/heartbeat (status, battery, GPS), GET /api/nodes/{id}/commands (command queue), GET /api/updates (OTA firmware), and GET /api/metrics (Prometheus). Compliance exports come from the portal as signed PDF and CSV. Ingest is keyed, scoped and rate-limited; every call is tenant-isolated.
