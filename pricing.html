HOW WE QUOTE

Three rules

Published prices are the prices we charge.

No "Talk to sales" hidden tiers, no anchoring against an inflated list price. If a public tier fits, you can self-serve. If you need something custom, we'll quote it from the same unit economics.

Pilots are short, scoped, and paid only if they earn it.

Two-week pilot, one fleet or one webhook source, a written success measure. If it doesn't meet the measure, you don't pay. We won't run an open-ended trial — that wastes both sides' time.

Annual ≠ aggressive discount. It's predictability.

Annual saves ~10% on Team / Pro, mostly to reduce billing churn. We won't run a quarterly "negotiation theatre" cycle to extract more from accounts that already pay fairly.

ALL THREE

Running more than one product?

No bundle SKU. Each product stands on its own value. If you run Acheron Vault, Blackglass, and Charon Gate together, support and onboarding are coordinated — which is the actual operational benefit.
